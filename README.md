# TEAM-60: HugsForBugs

## IMPORTANT

> Create a file `firebaseConfig.js` in /src directory and add the credentials of firebase project in the following way (Note that this **should not** be pushed to the GitHub).

```js
export const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
};
```
